# JSONL Viewer

Local offline copy of the JSONL viewer from [HamsterBase Tasks](https://tasks.hamsterbase.com/jsonl-viewer/).

Live site: [railgun.tech/jsonl-viewer](https://railgun.tech/jsonl-viewer/)

## Usage

Open [index.html](/Users/maemolee/GitHub/jsonl-viewer/index.html) in a browser.

## Notes

- The page is self-contained in a single HTML file.
- Analytics and Google Fonts were removed so it can run offline without external requests.
- PWA support is included via `manifest.webmanifest` and `service-worker.js`.
